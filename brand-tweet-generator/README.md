# AI Brand Tweet Generator

This tool generates 10 on-brand tweets based on brand information.

## Features

- Brand voice analysis
- AI-generated tweets
- Multiple tweet styles
- Simple UI

## Tech Stack

- Streamlit
- OpenAI API
- Prompt Engineering

## How It Works

1. User inputs brand information
2. AI analyzes the brand voice
3. AI generates tweets matching that voice

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run app:

streamlit run app.py
